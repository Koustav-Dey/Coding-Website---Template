C was originally developed at Bell Labs by Dennis Ritchie between 1972 and 1973 to make utilities running on Unix.

In 1989, C++ 2.0 was released, followed by the updated second edition of The C++ Programming Language in 1991.

Java was originally developed by James Gosling at Sun Microsystems and released in 1995 as a core component of Sun Microsystems' Java platform.

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991.

Ruby is an interpreted, high-level, general-purpose programming language. It was developed in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan.

PHP is a popular general-purpose scripting language that is especially suited to web development. It was originally created by Rasmus Lerdorf in 1994.

In November 1996, Netscape submitted JavaScript to ECMA International. This led to the official release of the first ECMAScript language specification in June 1997
